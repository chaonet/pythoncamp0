引用issue

同一仓库的issue commit message里面使用 #加上issue号 ，比如#8, github会自动关联issue 8跟这个commit.

其他仓库的issue 要链接到其他仓库的Issue，就使用user_name/repo_name#ISSUE_NUMBER的方式，例如tiimgreen/toc#12。

关闭issue

随着commit关闭一个issue,在confirm merge的时候可以使用以下命令来关闭相关issue:

fix #xxx
fixes #xxx
fixed #xxx
close #xxx
closes #xxx
closed #xxx
